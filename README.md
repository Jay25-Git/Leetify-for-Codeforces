# Awesome Leetify for Codeforces

> A community awareness page for **Leetify** — a free Chrome extension that turns the Codeforces problem page into a single, focused workspace (editor, testcases, notes, and analytics, all in one pane).

This repo isn't the extension itself — it's an unofficial page to help more people discover it, since a lot of Codeforces users still don't know it exists.

One toggle turns the default page into a split-pane workspace with a VS Code–style editor, notes, and analytics built in.

## Why This Repo Exists

Solving a Codeforces problem normally means juggling several tabs: the problem statement, your local editor, a place to run test cases, and maybe a notes doc. **Leetify** collapses all of that into one page, without changing what people already like about Codeforces' classic UI (you can toggle back to the original view any time).

## Features

| Feature | What it does |
|---|---|
| Split-pane workspace | Problem statement + editor side by side, resizable |
| VS Code–style editor | Syntax highlighting, line numbers, active-line highlight |
| Testcase panel | Run sample tests or add custom ones inline |
| One-click submit | Submit without leaving the editor pane |
| Difficulty & tags | Color-coded ratings and topic tags across the site |
| Per-problem notes | Notes tied to each problem, searchable, pinnable, tag-filterable |
| Practice analytics | Streaks, solve-density heatmap, rating trends, topic depth, readiness |
| Smart language selector | Pins your frequently used compilers to the top |
| Dark mode | Built for long contests and late-night practice |
| Site-wide | Works on problemset, contests, gym, and mashup pages |

## Install

1. Open the [Chrome Web Store listing for "Leetify for Codeforces"](https://chromewebstore.google.com/detail/leetify-for-codeforces/efdidgeaehmaiobeldfiomlnhjnneghl).
2. Click **Add to Chrome**.
3. Open any Codeforces problem — the workspace appears automatically.

Free, no account required.

**Demo / project site:** https://leetify-dun.vercel.app/

## Status

Leetify is early-stage and moving fast (v0.1.5 as of this writing), with new features often coming directly from user suggestions on Codeforces blog threads. If you're on the fence about the modern layout vs. classic CF look, note that switching between the two is built in.
